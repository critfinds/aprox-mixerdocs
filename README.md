# AproxMixer Documentation

Welcome to the official documentation for **AproxMixer**, a privacy-focused Web3 application that leverages zero-knowledge proofs to enable unlinkable transactions on Ethereum and EVM-compatible networks.

---

## Overview

AproxMixer allows users to:

- Deposit and withdraw assets with privacy.
- Interact with zero-knowledge proof-based smart contracts.
- Learn about the protocol, architecture, and best practices.
- Explore a comprehensive API reference and tutorials.

This documentation site is built using:

- **Vite** — for fast development and build times.
- **React** — for a modern and interactive UI.
- **Tailwind CSS + shadcn/ui** — for responsive and customizable styling.
- **React Router v6** — for page navigation.
- **React Query** — for efficient data fetching and caching.
